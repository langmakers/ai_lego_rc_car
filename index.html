Project Overview

End-to-End Autonomous RC Car with Embedded LLM & ROS2

This project aims to build a fully autonomous RC car platform using a combination of hardware and AI software. Starting with a Lego Defender 42110 chassis, we integrated servo and DC motors, controlled by Arduino Uno and Raspberry Pi 5. The system supports remote control via HTTP and I2C, and features a mobile app for voice-based control using Whisper and LLM APIs. The latest milestone introduces ROS2 and LiDAR for SLAM-based autonomous navigation. All control and collaboration are managed via email communication.

Technologies Used

Lego Defender 42110 Arduino Uno Raspberry Pi 5 Servo/DC Motor Whisper LLM API ROS2 LiDAR

Project Milestones

  1. RC Car Preparation Done
    • Installed motors on Lego Defender 42110
    • Checked servo/DC motor control methods
    • Tested motor control with Arduino Uno
  2. Remote Control Done
    • Controlled Arduino (Uno) motors via I2C from Raspberry Pi (RPI5)
    • Integrated motor control via HTTP protocol on Raspberry Pi
  3. Mobile Control App Done
    • STT using Whisper-based microphone signal
    • Built remote MCP Server on Raspberry Pi
    • Converted voice commands to control signals via LLM API
  4. ROS2 Adoption In Progress
    • SLAM with LiDAR sensor

Key Features

Precise Motor Control

Servo and DC motors are precisely controlled via Arduino Uno and Raspberry Pi, enabling accurate steering and driving.

Remote & Mobile Control

Supports remote operation through HTTP and I2C, and mobile app-based control for flexible driving experiences.

Voice Command Integration

Whisper and LLM APIs convert voice commands into real-time control signals for hands-free operation.

Autonomous Navigation (SLAM)

ROS2 and LiDAR enable SLAM-based mapping and autonomous driving capabilities.
