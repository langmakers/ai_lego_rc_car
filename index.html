Project Overview

Autonomous RC Car Implemented with AI Technology

This project is an autonomous RC car based on Raspberry Pi and Arduino. It recognizes the surrounding environment in real-time through computer vision technology, and determines the optimal driving path using deep learning models.

Technologies Used

Python OpenCV TensorFlow Raspberry Pi Arduino ROS

Key Features

Computer Vision

Recognizes the surrounding environment in real-time through cameras and detects obstacles.

Machine Learning

Learns and determines optimal driving paths using deep learning models.

Path Planning

Generates maps and plans efficient routes through SLAM algorithms.

Safety System

Ensures safe driving with emergency stop functions and collision prevention systems.

Remote Control

Control and monitor the RC car remotely through smartphone apps.

Real-time Monitoring

Collects and analyzes driving data and sensor information in real-time.
